# SlateOS Roadmap

## Vision

Create a fast, private, beautiful, and highly customizable open-source operating system.

The goal:

> A computer experience where the user is always in control.

# Long-Term Goals

SlateOS aims to become:

* A daily driver operating system
* A privacy-focused alternative
* A developer-friendly platform
* A community-owned project

---

# Development Rule

Features should be built in order of importance:

1. Foundation
2. Stability
3. Performance
4. Security
5. User experience
6. Advanced features

Do not sacrifice the foundation for flashy features.
