# About SlateOS

SlateOS is an open-source operating system designed around one idea:

> **Your computer belongs to you.**

SlateOS combines:

* The elegance of macOS
* The customization of Linux
* The compatibility of Windows
* The privacy philosophy of open-source software

No forced accounts.
No advertisements.
No unnecessary tracking.

Just a clean, powerful operating system.

---

# Features

## Unlimited Customization

SlateOS is built to let users completely personalize their system.

Customize:

* Wallpapers
* Video wallpapers
* HTML/CSS wallpapers
* Themes
* Icons
* Fonts
* Animations
* Window layouts
* Sounds
* Boot screens
* Desktop environments

Your computer should feel like yours.

---

# 🔒 Privacy First

Privacy is not an extra feature.

It is the foundation.

SlateOS aims to provide:

* No telemetry by default
* No hidden tracking
* No mandatory cloud accounts
* Local-first processing
* Transparent permissions
* Full user control

Your data stays on your device.

---

# Lightweight Performance

SlateOS is designed to be fast.

Goals:

* Low RAM usage
* Fast startup times
* Minimal background services
* Efficient resource management
* Smooth performance on old and new hardware

Every process should have a reason to exist.

---

# Slate Guardian

A built-in privacy-focused security system.

Features:

* Real-time malware detection
* Behaviour monitoring
* Application sandboxing
* Download scanning
* Permission controls
* Local security analysis

Security without surveillance.
